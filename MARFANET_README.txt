# MarFaNet Android App

This APK contains the MarFaNet VPN application with the following features:
- Auto-ping servers every 10 minutes
- Auto-update subscriptions every 30 minutes
- Auto-connect to server with lowest ping
- Daily routing rules updates
- Support for various protocols including VMess, VLESS, Trojan, Shadowsocks, Hysteria, and REALITY

## Important Notes

This APK is a demonstration package. For a fully functional app, you'll need to:

1. Import the source code into Android Studio
2. Build the project with proper Android SDK tools
3. Sign the APK with your own keystore

## How to Install

1. Copy the APK to your Android device
2. Enable "Install from Unknown Sources" in your device settings
3. Tap the APK file to install
